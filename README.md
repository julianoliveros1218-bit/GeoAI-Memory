# GeoAI para la detección de cambios urbanos

## Aplicación de inteligencia artificial geoespacial en imágenes satelitales, catastro y planeación territorial

Este repositorio contiene una página web académica desarrollada en **GitHub Pages** sobre el uso de **GeoAI** para la detección de cambios urbanos mediante imágenes satelitales, sensores remotos, sistemas de información geográfica e inteligencia artificial aplicada al análisis territorial.

El proyecto explica cómo la inteligencia artificial geoespacial puede apoyar la identificación de nuevas construcciones, expansión urbana, pérdida de vegetación, cambios en el uso del suelo y transformaciones físicas del territorio.
